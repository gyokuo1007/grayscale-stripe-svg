# svg-stripe-generator 
グレースケール画像を読み込み、輝度に応じて水平ストライプ状のSVGを生成するPythonツールです。   ブロック単位の明度解析により、暗い領域ほど密度の高いストライプが描画されます。  ユーザーは画像選択後、キャンバスサイズや縦横比の保持をGUIで指定可能。   さらに「パスを結合」オプションにより、全ての線を一本化されたSVGパスとして出力できます。  ## 特徴 - Tkinterベースの直感的なGUI - 日本語ファイルパスにも対応 - 明度によるストライプ密度制御 - SVG出力のマージ処理（連続した線を結合） - ファイル名の衝突回避機能
